# 编程题 网易云音乐微信小程序版接口文档

## 接口说明

### 请求基准地址

```text
https://netease-cloud-music-api-sand-three.vercel.app
```

## 1.获取首页轮播图

请求地址:

```text
/banner
```

请求参数:

```text
type:资源类型,对应以下类型,默认为 0 即PC
0: pc

1: android

2: iphone

3: ipad
```

响应示例:

## 2.获取推荐歌单

请求地址:

```text
/personalized
```

请求参数(可选):

`limit`: 取出数量 , 默认为 30 (不支持 offset)`

响应示例:

```json
{
  "hasTaste": false,
  "code": 200,
  "category": 0,
  "result": [
    {
      "id": 5000331729,
```

## 3.获取默认热搜关键词

请求地址:

```text
/search/default
```

请求参数: 无

响应示例:

```json
{
  "code": 200,
  "message": null,
  "data": {
    "showKeyword": "华晨宇 最近很火哦",
    "realkeyword": "华晨宇",
    "searchType": 1018,
    "action": 0,
    "alg": "alg_query_hot",
    "gap": 0,
    "source": null
  }
}
```

## 4.获取热搜榜

请求地址:

```text
/search/hot/detail
```

请求参数: 无

响应示例:

```json
{
  "code": 200,
  "data": [
    {
      "searchWord": "你的样子",
      "score": 1583394,
```
